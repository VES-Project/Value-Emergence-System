## 自由エネルギー原理

生物システムや認知システムが、環境からの予測誤差（自由エネルギー）を最小化するように振る舞うという原理です。

```typescript
function calculateFreeEnergy(prediction: number, actual: number): number {
  // これは単なる例です
  const error = prediction - actual;
  return error * error;
}
```

<Alert type="warning">注意: このコードは説明のための疑似コードです。</Alert>

## その他の原理

- **情報理論:** 価値の伝達や変化を情報の流れとして捉えます。
- **熱力学:** 社会構造の安定性や変化をエネルギーの観点から考察します。
